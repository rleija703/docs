# üîç Search

The **Search** feature gives you access to over **6,500 U.S. optionable tickers** directly inside Gextron. You can pull up any security in seconds using ticker, company name, or security name.

<Callout type="info">
Search is **plan-dependent**:  
- **Free Plan** ‚Üí limited to **SPY, QQQ, IWM** only  
- **Hero Plan** ‚Üí access to the **full universe** (6,500+ tickers)  
- **Pro Plan** ‚Üí full universe + all Pro features (Tape, advanced targets, etc.)
</Callout>

---

## How to Open Search

<Tabs>
  <Tab title="Navigation Bar">
    - On the far **left navigation bar**, click on the **Search tab**.  
    - A dialog opens ‚Üí start typing the **ticker symbol**, **company name**, or **security name**.  
    <Frame>
      <img src="https://alfatoro.com/wp-content/uploads/2025/09/search-navigation-icon.png" alt="Search from navigation bar" />
    </Frame>
  </Tab>

  <Tab title="Keyboard Shortcut">
    - Press **Ctrl + K** (Windows) or **Cmd + K** (Mac).  
    - The **Search dialog** pops up instantly.  
    - Begin typing and results will appear in real time.
    <Frame>
      <img src="https://alfatoro.com/wp-content/uploads/2025/09/search-navigation-icon.png" alt="Search from navigation bar" />
    </Frame>
  </Tab>

  <Tab title="Inline Typing">
    - Simply start typing a **ticker symbol** or **company name**.  
    - Gextron will automatically detect input and open the **Search results** dialog.  
    - No extra clicks required.  
  </Tab>
</Tabs>

## What You Can Search

<Frame>
  <img src="https://alfatoro.com/wp-content/uploads/2025/09/gextron-search-dialog.png" alt="Gextron search dialog opened" />
</Frame>

- **Ticker symbol** (e.g., `AAPL`)  
- **Company name** (e.g., `Apple`)  
- **Security name** (e.g., `Apple Inc. Common Stock`)  

<Callout type="info">
Search is designed to be **fast and forgiving** ‚Äî partial matches and common abbreviations still surface results.
</Callout>

## Why It Matters

- Quickly navigate the **entire options universe** (6,500+ names).  
- No need to memorize tickers ‚Äî type company names instead.  
- Integrated with the **Pro Dashboard** ‚Üí once selected, the ticker loads instantly into Key Levels, Price Targets, Option Flow, or Tape.
